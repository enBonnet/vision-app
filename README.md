## My Hooks have Vision

Demo de la charla con el mismo nombre.

Para ejecutar el proyecto:

```bash
git clone https://github.com/enBonnet/vision-app
cd vision-app
npm install
cp .env.sample .env
#Completar las key/variables requeridas

#Ejecutar el servidor
npm run server
#Ejecutar el cliente
npm start
#Listo! A jugar!
```
